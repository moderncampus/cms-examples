# Policy Manual

The code here gives a way to build out a Policy section of a University's site. The code contains three page types, the Table of Contents page, a Policy Section's Home page, and Policy Article pages.

Users who wish to implement this code should have a basic understanding of XSL, Modern Campus' DMC, and the implementation skeleton. 

Every Policy Section's Home (Index) Page has the following:
 - Main Content Region
 - Automated DMC generated list of links to the section's policy article pages
 - Additional Content Region
 - Multiedit section to provide information to the DMC aggregation about the section's Policy

Every Policy Article Page has the following:
 - Main Content Region
 - Right Side Listing of the Section's fellow Policy Articles
 - Multiedit section to provide information to the DMC aggregation about the Policy Article

Every Policy Table of Contents Page has the following:
 - Main Content Region
 - Automated DMC generated list of Content and links realted to/generated by all Policy Subsections
 - Additional Content Region

Other Content provided includes
- TMPL/TCF combination for each of the three page types
- Data PCF for the policies multiedit aggregation
- XSL for each page's XML output, with an updated/modified ou:get-item() function call to allow for the content aggregation necessary for the Table of Contents Page

## Dependencies

This code requires the DMC and subsequent files that are included/necessary for the DMC.
